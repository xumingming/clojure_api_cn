run-all-tests
______________

| **函数**
| **用途: (run-all-tests)
      **(run-all-tests re)**


运行所有名字空间下面的所有测试代码；打印结果。可选参数是一个正则表达式；当传入这个参数的时候，只有名字符合这个正则表达式(利用re-matches)的测试会被执行。

添加于Clojure 1.1
